Read the tags: Start = high intent + real volume, launch here. Keep on = good intent, thin volume — run on phrase/exact at low bids. Copy/audience = near-zero as literal searches; use these as ad copy, landing-page sections and audience segments, not keywords (a parent mid-divorce searches the symptom, not "…for kids going through divorce").

Worth testing in Keyword Planner too: "social skills classes for kids / online", "virtual social skills group for kids", "online social skills classes for autism / ADHD", "social skills training for kids", "Minecraft club for kids with autism", plus teen variants if the program serves older kids. Day-one negatives: free, PDF, worksheet, jobs, salary, certification, ABA, "therapist near me", and city names (to stay national).
